# 📚 MyLibrary
A Library Managment System 

# 🚀 Features
👤 Authentication & Users
- Registracija korisnika
- Login (JWT autentikacija)
- Role-based autorizacija (user / admin)
- Uređivanje profila
- Upload korisničkog avatara (Multer)
  
📚 Books
- Registracija korisnika
- CRUD operacije (admin)
- Upload slike knjige
- Filtriranje po žanru
- Automatsko praćenje dostupnih primjeraka
- Sprječavanje posudbe ako nema dostupnih kopija
  
🏷 Genres
- Dodavanje žanra (admin)
- Brisanje žanra (admin)
- Filtriranje knjiga po žanru

🔄 Loans
- Posudba knjige (user)
- Evidencija svih posudbi (admin)
- Povrat knjige
- Sprječavanje duple aktivne posudbe iste knjige
